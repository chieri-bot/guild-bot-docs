# 代码模拟运行

- `/ch run 语言 代码`----指令例:`/ch run py print("我是python")`



## 支持的语言

- PHP - `php`
- python3 - `py`
- python2 - `py2`
- Java - `java`
- C - `c`
- C++ - `cpp`
- Ruby - `rb`
- C# - `cs`
- Scala - `scala`
- Erlang - `erl`
- Perl - `pl`
- Bash - `sh`
- RUST - `rs`
- Swift - `swift`
- Golang - `go`
- Node.js - `njs`
- Lua - `lua`
- Pascal - `pas`
- Kotlin - `kt`
- TypeScript - `ts`
- VB.NET - `vb`
- R - `r`
