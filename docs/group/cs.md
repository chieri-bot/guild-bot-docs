# CS点数相关



## 查看个人信息

- 查看cs点数和经验值

- 指令: `/ch i`



## 早/晚安

:::tip

目前仅有这一种方式可以**稳定**获取CS点数和经验值

:::

- `chieri早`， `chieri晚`



## 抽奖

:::tip

消耗5CS抽奖，中奖后可获得奖池内所有CS（一夜暴富！测试欧气的时刻到了！）

:::

- 指令: `/抽奖` 



## 碰碰运气

:::tip

消耗指定数量CS点碰运气，亏还是赚由上天决定~

注: 若输入负数, 则会投入您的cs总数的五分之一

:::

- 指令: `/ch cs 数字`



## 转账

- 指令: `/ch pay QQ号 数量`



## 使用兑换码

:::tip

在某些特殊的日子, 会发送兑换码。关注chieri获取最新资讯!

:::

- `/ch con 兑换码`
