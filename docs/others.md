# 杂项

- 除了主要的音游模块, chieri也有一些小功能



## 多语言

### 支持的语言

:::tip

Bot可以支持任意语言, 若您有能力且愿意帮助chieri进行多语言适配, 欢迎联系开发者~

:::

- `zh-cn` - 简体中文 **(默认)**
- `en` - English (未完全翻译)



### 切换语言

- 指令: `/set lang 目标语言`



## 表情包生成

### 五千兆

- 指令: `/五千兆 这是红字 这是白字`

:::tip 附加

您可以使用`-colorful`参数生成炫彩图片, 指令例: `/五千兆 这是红字 这是彩字 -colorful`

:::



## 天气查询

- 指令: `/天气 地名`
