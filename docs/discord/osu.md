# OSU!

- [OSU!](https://osu.ppy.sh/) is an rhythm game, please make sure you have an `OSU!` account before using.



## Bind Account

- Command: `/osu bind yourID`



## Recent Grades

- Command: `/osu recent`



## User Profile

- Command: `/osu info [mode]`
  - `mode` is optional. `0 - osu`, `1 - taiko`, `2 - fruits`, `3 - mania`
