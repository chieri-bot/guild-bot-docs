# Others

- In addition to the main rhythm game module, chieri also has some other features.



## Multi Language

:::tip

Bot可以支持任意语言, 若您有能力且愿意帮助chieri进行多语言适配, 欢迎联系开发者~

:::

- `zh-cn` - 简体中文
- `en` - English (incomplete)



### Switch Language

- Command: `/set lang`



## Image Generation

### 5000choyen

- Command: `/5000choyen text1 text2 [-colorful]`



## 天气查询

- 指令: `/weather 地名`
